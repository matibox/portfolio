3. MY PROJECTS

DESIGN 2021.

I made this website at the end of 2021. It was for a school project. I wanted to try GSAP - an animation library for JS, especially the new scroll trigger option. The site is fully responsive and features the bulkiest trends in web design in 2021 and some cool scroll animations. It also includes a horizontal scroll section.

TIC-TAC-TOE.

Yes, I know what you think. Is this guy putting tic-tac-toe on his portfolio? Well, the answer is yes but actually no! It is tic-tac-toe but with different rules. After each player puts three marks game mode changes. Now they need to move the marks to empty cells instead of putting new ones. The principle makes the game way more interesting than the original one. I'm looking to add other features in the future. You need to test it with a friend!

TODO APP.

The todo app, as a project taken from frontendmentor.io, is more than a todo app. It features a lot of unusual things apart from being just a simple list. Users can: view the website correctly on every device, add new todos to the list, mark todos as complete, delete todos from the list, filter by all/active/completed todos, clear all completed todos, toggle between light and dark mode, drag and drop to reorder todos. I also added one feature myself - you can also add links to the todo.
